sqlREST Developer Guide

Building

To build sqlREST from source you need only Java SDK an the source from cvs at soureforge. The buildtool ant is provided with the sqlREST project. Just cd to the bin directory and choose the distribution target:

cd bin build distribution

After these steps you will find an distribution inside the dist directory.
